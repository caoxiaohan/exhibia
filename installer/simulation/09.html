Table Prefix

The prefix prx_ is accepted. Proceed to the next step.
The table prefix is optional, but highly recommended. Keep the box empty if you wish not to set a table prefix.
You cannot use # as prefix value, choose another!

The prefix is used to prevent collision with other database tables, please enter few characters to uniquely identify your tables from the others.

Note that the symbol _ will be added automatically at the end of the prefix. If added manually, it will not be repeated.
