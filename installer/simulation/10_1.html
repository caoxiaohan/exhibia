Install Database Tables

You are about to install tables on usethisdb though a valid install has been successfully done already.
 
You can go back to Database Selection and select a databases that has a valid installation.
You have not specified a table prefix, and the database usethisdb contains 6 table(s).
 
It is highly recommended and there is a much lower risk of table collision if you specify a table prefix.

The database usethisdb contains the following tables:

hj_activity hj_blog hj_category
hj_comment hj_contact hj_user

Be completly sure that this is the database you want to use, even though it is not empty before continuing with installation.
